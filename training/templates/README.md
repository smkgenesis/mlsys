# templates

Decision and design templates for the training/model-optimization branch.

This folder is separate from the concept notes in [training/README.md](/Users/minkyu/Documents/mlsys/training/README.md).

Use this folder for:

- bottleneck-to-technique mapping checklists,
- optimization design templates,
- and compact reconstruction aids for exam or interview-style reasoning.

Do not use this folder for:

- full lecture summaries,
- framework-specific production code,
- or highly detailed kernel implementations.

Working rule:

```text
notes explain the optimization dimension;
templates capture the smallest reusable decision pattern.
```
