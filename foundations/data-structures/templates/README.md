# templates

Java hand-coding templates for the data-structures track.

This folder is separate from the concept notes in [foundations/data-structures/README.md](/Users/minkyu/Documents/mlsys/foundations/data-structures/README.md).

Use this folder for:

- minimal handwritten skeletons,
- operation templates that can be reconstructed under exam pressure,
- invariant-first coding aids,
- and small mutation notes for likely variations.

Do not use this folder for:

- full lecture summaries,
- polished library-style implementations,
- or structures that have not yet appeared in the lecture materials.

Working rule:

```text
notes explain why the structure works;
templates capture the smallest code skeleton we can reliably rebuild by hand.
```
