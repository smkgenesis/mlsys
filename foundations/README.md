# foundations

Basic computer science foundations only.

Belongs here:
- CS-first reasoning habits,
- recursion,
- stacks,
- core data-structure or algorithmic thinking before specialization,
- and foundational concepts that are primarily part of basic CS study.

Does not belong here:
- mathematics-first material, which belongs in `math/`,
- cross-domain study methods, which belong in `roadmap/`,
- architecture- or systems-specific material better placed in `architecture/`, `gpu/`, or `cuda/`.
